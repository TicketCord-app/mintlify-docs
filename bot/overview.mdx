## What is TicketCord?

TicketCord is a white-label Discord ticket bot that lets you run your own fully customized support system. Each bot you create is completely independent with its own:

- Name and avatar
- Embed colors and branding
- Configuration and settings
- Ticket data and transcripts

## Key Features

### Ticket Management

- **Custom panels** - Create ticket panels with buttons or dropdown menus
- **Multiple categories** - Organize tickets by type (support, sales, etc.)
- **Claiming system** - Staff can claim tickets to avoid duplicated effort
- **Auto-close** - Automatically close inactive tickets

### AI Features

<Info>AI features require a Pro or Enterprise plan.</Info>

- **Atlas AI** - Dashboard assistant for analytics and insights
- **Knowledge Base** - Auto-suggest answers from your documentation
- **Smart Routing** - Automatically assign tickets to the right team
- **Priority Detection** - AI-powered ticket prioritization

### Customization

- **Custom embeds** - Design ticket panels with the embed builder
- **Application forms** - Collect information before ticket creation
- **Custom footer** - Remove or customize the bot's footer text
- **White-label** - Full branding control on Enterprise plan
